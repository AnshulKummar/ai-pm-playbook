# Privacy Policy

**Last updated:** 2026-03-14

## Overview

This repository contains educational documentation (a guide/playbook). It is not a hosted service, application, or platform. There is no account creation, data collection, or telemetry.

## Data Collection

This project does **not**:

- Collect, store, or process any personal data
- Use analytics, tracking, or telemetry
- Require account creation or authentication
- Send data to any external service

## User Responsibilities

When using hooks, scripts, or configurations from this guide:

- **Do not commit sensitive data.** Never include API keys, tokens, passwords, or personally identifiable information in files you commit to any repository.
- **Review before sharing.** If you share your CLAUDE.md, settings.json, or hook scripts publicly, review them for any sensitive information first.
- **Issue reports.** When filing issues or contributing, do not include proprietary code, internal URLs, credentials, or confidential business information.
